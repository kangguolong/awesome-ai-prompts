# Tool-Specific Prompts

Use this folder only for prompts that depend heavily on a specific AI product or workflow environment.

Examples include prompts that rely on:

- ChatGPT custom GPT behavior
- Claude artifacts
- Cursor repository context
- Perplexity search behavior
- GitHub Copilot coding environment assumptions

Prefer general workflow prompts outside this folder when the prompt can work across multiple AI tools.

## Suggested Subfolders

```text
chatgpt/
claude/
cursor/
gemini/
perplexity/
github-copilot/
```

Create subfolders only when there are enough prompts to justify them.
