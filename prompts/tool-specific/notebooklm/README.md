# NotebookLM Prompts

Prompts in this folder are designed for workflows that use NotebookLM as a research notebook built from selected sources.

Use these prompts when the quality of the answer depends heavily on source selection, source reliability, and structured synthesis across uploaded materials.

## Recommended Workflow

1. Define the strategic decision clearly.
2. Discover and screen sources before adding them to the notebook.
3. Add only high-quality, decision-relevant sources.
4. Use NotebookLM to synthesize across the selected source set.
5. Validate important claims against primary sources before making the final decision.

## Prompts

| Prompt | Use Case |
|---|---|
| [`strategic-source-discovery.md`](strategic-source-discovery.md) | Find and evaluate sources to add to a strategic decision-making notebook |
