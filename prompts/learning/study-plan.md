# Study Plan

## Use Case

Use this prompt to create a structured study plan for learning a concept, skill, tool, or domain.

## Best For

- ChatGPT
- Claude
- Gemini

## Prompt

```text
Create a practical study plan for learning the topic below.

Topic:
[TOPIC]

Current level:
[CURRENT_LEVEL]

Goal:
[GOAL]

Available time:
[TIME_AVAILABLE]

Constraints:
[CONSTRAINTS]

Return the plan in this format:
1. Learning objective
2. Prerequisites
3. Study roadmap by stage
4. Weekly schedule
5. Practice exercises or projects
6. Recommended checkpoints
7. Common mistakes to avoid
8. How to measure progress

Prioritize practical understanding and application over passive reading.
```

## Variables

| Placeholder | Description |
|---|---|
| `[TOPIC]` | Skill, topic, tool, language, or domain to learn |
| `[CURRENT_LEVEL]` | Beginner, intermediate, advanced, or specific background |
| `[GOAL]` | Desired outcome or capability |
| `[TIME_AVAILABLE]` | Hours per week, deadline, or learning horizon |
| `[CONSTRAINTS]` | Budget, preferred learning style, tools, exclusions, or required depth |

## Example Usage

```text
Topic:
System design for backend engineers.

Current level:
Intermediate backend developer.

Goal:
Prepare for senior engineering interviews and improve architecture judgment.

Available time:
5 hours per week for 8 weeks.

Constraints:
Focus on practical trade-offs, reliability, scalability, and distributed systems basics.
```

## Expected Output

A staged learning plan with practical exercises, checkpoints, and progress measures.

## Notes

- Add a target deadline when possible.
- Include hands-on projects for technical subjects.
- Ask for adjustments if the first plan is too broad or too shallow.
