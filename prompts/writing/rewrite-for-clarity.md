# Rewrite for Clarity

## Use Case

Use this prompt to rewrite text so it is clearer, more concise, and better aligned with a target audience.

## Best For

- ChatGPT
- Claude
- Gemini

## Prompt

```text
Rewrite the text below for clarity, structure, and readability.

Audience:
[AUDIENCE]

Goal:
[GOAL]

Tone:
[TONE]

Constraints:
[CONSTRAINTS]

Text:
[TEXT]

Return the result in this format:
1. Rewritten version
2. Key improvements made
3. Optional alternative version if useful

Preserve the original meaning. Do not add unsupported claims.
```

## Variables

| Placeholder | Description |
|---|---|
| `[AUDIENCE]` | Intended reader |
| `[GOAL]` | Purpose of the text |
| `[TONE]` | Desired tone, such as concise, executive, friendly, technical, or persuasive |
| `[CONSTRAINTS]` | Length, formatting, vocabulary, exclusions, or style rules |
| `[TEXT]` | Original text to rewrite |

## Example Usage

```text
Audience:
Engineering manager.

Goal:
Explain the project risk clearly and professionally.

Tone:
Direct and concise.

Constraints:
Keep under 150 words. Do not sound defensive.

Text:
[Paste draft here]
```

## Expected Output

A cleaner version of the original text plus a brief explanation of what changed.

## Notes

- Good for emails, documentation, internal updates, and summaries.
- Add examples when a specific style is required.
- Review sensitive or legal text carefully before sending.
