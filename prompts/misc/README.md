# Miscellaneous Prompts

Use this folder for useful prompts that do not yet fit a major category.

A prompt should stay here only temporarily. If several related prompts accumulate around the same workflow, consider creating a dedicated category.

Before adding a prompt here, check whether it fits one of the existing categories:

- `coding/`
- `research/`
- `business/`
- `writing/`
- `productivity/`
- `decision-making/`
- `learning/`
- `tool-specific/`
